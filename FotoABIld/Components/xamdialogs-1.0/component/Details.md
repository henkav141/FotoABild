# XamDialogs

Custom dialogs for picking dates and from simple lists for Xamarin.iOS

 * Closely resembles new style iOS 7+ AlertViews
 * Customisable
 * Date and Picker view in alert dialog
 * Extensible, create your own custom dialogs
 * Styleable
